# Project Buenas Viagens ✈️

Este projeto é o desenvolvimento de uma página web responsiva para uma agência de viagens.

A atividade faz parte da atividade avaliativa v3 da disciplina TEcnologias da Internet  na facuculdade FB UNI.


##  Tecnologias Utilizadas

-HTML5
-CSS3
-JavaScript
-Git & GitHub
